**markdown语法**

> （1）级别标题 
>
> 一个井号加空格 就是一级标题，两个井号加空格 就是二级标题，以此类推
>
> （2）代码块
>
> 两组三个反引号中间写代码，回车。如下：
>
> \```
>
> a = 1
>
> \```
>
> “```”是数字键1前面那个键，切换成英文格式打出来就是
>
> （3）引用
>
> 大于号加空格 后面写内容即可
